Projects

Bright.ED

Bright.ED is an educational website designed to make learning interactive and accessible for students of all ages.
It offers a variety of courses, quizzes, and resources to help users enhance their knowledge and skills. The platform features user-friendly navigation and personalized learning paths to support effective education.

E-Commerce

This E-Commerce platform is a client-side website designed for selling metal products and manufacturing machines.
It provides an easy-to-use interface for browsing, selecting, and purchasing industrial goods. The site features detailed product listings, secure checkout, and efficient navigation to streamline the buying process for customers in the manufacturing sector.

Gemini-clone

Gemini-clone is a web-based application inspired by Google's Gemini AI. It simulates conversational AI interactions, allowing users to ask questions and receive intelligent responses. The project demonstrates modern web development techniques and showcases the integration of AI-like features in a user-friendly interface.
